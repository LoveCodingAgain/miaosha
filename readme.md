#### 高并发秒杀系统
##### 一. 项目开发背景
1. 基于SpringBoot框架在传统的秒杀项目上面进行支持高并发的优化.做压测进行先后对比.在单台和集群部署下的系统的QPS值的对比. 
2. 对Redis/RabbitMQ有更加深入的学习.
 ---
